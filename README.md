# tool_browse_users_classic

## Overview
`tool_browse_users_classic` is a simple Moodle plugin that brings back the classic (i.e. before 
Moodle 4.3) "Browse list of users" page.

## Installation
1. Download the plugin and extract it into the `tool` directory of your Moodle installation.
2. Navigate to the Site Administration > Plugins > Install plugins page.
3. Follow the on-screen instructions to complete the installation.

## Usage
Once installed, you can access the plugin by navigating to:
http://yourmoodlesite/admin/tool/browse_users_classic/index.php
You should see the old "Browse users" page displayed.
